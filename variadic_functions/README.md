
# Fonctions Variadiques

## Description
Ce dossier contient des exercices sur les **fonctions variadiques** en C, permettant de passer un nombre variable d'arguments à une fonction.

## Concepts couverts
- `stdarg.h` : macros pour manipuler les arguments variables
- `va_list`, `va_start()`, `va_arg()`, `va_end()`
- Fonctions avec nombre d'arguments indéterminé

## Fichiers
- Exercices pratiques d'implémentation
- Exemples de fonctions variadic

## Compilation
```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o programme
```
